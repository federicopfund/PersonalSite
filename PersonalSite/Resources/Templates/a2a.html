Ejemplo JSON-RPC (message/send)

message/send · curl 
curl -X POST <* TemplateSlot["rpcUrl"] *> \
  -H 'Content-Type: application/json' \
  -d '{
    "jsonrpc": "2.0",
    "id": "1",
    "method": "message/send",
    "params": {
      "message": {
        "role": "user",
        "parts": [{ "kind": "data", "data": { "seed": 1, "depth": 3 } }],
        "messageId": "msg-1"
      }
    }
  }'

Transporte canónico A2A: application/json. En este despliegue con Wolfram Web Engine el cuerpo crudo JSON no se expone, así que el endpoint también acepta el mismo payload como application/x-www-form-urlencoded, o podés usar GET /a2a/run?seed=1&depth=3.
